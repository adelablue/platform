## File structure used
This project is using a feature module organization pattern. Please use the following, or similar, file structture:

* / (project root)
  * problemCategories/
    * both/
      * collections/
        * collection.js
        * schema.js
        * permissions.js
    * client/
      * templates/
        * templateName.html
        * templateName.js
    * server/
      * methods.js
      * publications.js
